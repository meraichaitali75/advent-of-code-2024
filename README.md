#Advent of Code 2024 - Day 2 Solution

This repository contains my solution for the **Day 2: Red-Nosed Reports** challenge from Advent of code 2024.

##Technical Approach
The solution is implemented in **TypeScript**, focusing on: 
* **Redability:** Using descriptive naming and a clean "ReportAnalyzer" utility.
* **Scalability:** Implementation of the "Problem Dampener" using a single-pass validation logic.
* **Functional Patterns:** Utilizing ".some()", ".filter()", ".map()" for idiomatic Javascript/TypeScript.

* ## How to Run
* 1. Ensure you have 'node' and 'ts-node' instaled.
  2. Clone the repo: 'git clone https://github.com/meraichaitali75/advent-of-code-2024.git'
  3. Run the script: 'ts-node day2_solution.ts'
